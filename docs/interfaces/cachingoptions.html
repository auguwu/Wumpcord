Optional engine

engine: AbstractEntityCache | "memory" | "no-op"
  • Defined in src/types.ts:215

The default entity cache engine to use, you can also override a specific entity's cache engine, for an example, if you don't need to cache channels, you can just set channels to a instance of NoopEntityCache.

NOTE: If you do not want to construct a new instance of an AbstractEntityCache engine, you can just specify the built in ones: 'memory' or 'no-op'
